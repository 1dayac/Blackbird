New SPAdes 3.7

Dear user,

We are happy to announce that we have just released new SPAdes 3.7.0. This version includes the following modifications:

- New truSPAdes â€“ an assembler for short reads produced by Illumina TruSeq Long Read technology;
- Better running time, less RAM consumption and improved results for BayesHammer error correction module;
- Improvements and bugfixes in dipSPAdes;
- Multiple improvements and bugfixes in repeat resolution and scaffolding modules;
- MismatchCorrector module now uses bwa-mem; several bugfixes are made.

Please note that SPAdes was designed for single-cell and standard bacterial and other small (e.g. fungal) data sets and was not intended for larger genomes (e.g. mammalian size genomes). Should you decide to use SPAdes for a larger genome, please be aware that you are doing it at your own risk.

SPAdes version 3.7.0 is released under GPLv2 and openly available for download from http://bioinf.spbau.ru/en/spades.

Your comments, bug reports, and suggestions are very welcomed. They will help us to further improve SPAdes. If you have any troubles running SPAdes, please provide us with the files params.txt and spades.log from the output directory. Address for communications: spades.support@bioinf.spbau.ru.
